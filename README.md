# project-web-1
Quản lý ký túc xá là việc kết hợp giữa quản trị con người và quản trị vận hành nhằm tạo ra một môi trường sống an toàn, lành mạnh, tiện nghi, hỗ trợ tối đa cho việc học tập và làm việc của người lưu trú. Trong thời đại số, công việc này thường được tối ưu hóa bằng các phần mềm quản lý thông minh.
